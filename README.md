Performed end-to-end data analysis and machine learning on a multi-year public transport ridership dataset spanning multiple transport modes (MetroBus, rail, etc.). 
Cleaned and engineered features including COVID-19 period classification (Pre-2020, 2020–2021, Post-2021), weekday flags, and 3-month rolling averages grouped by 
mode and day type. Conducted exploratory data analysis through descriptive statistics, time-series trend lines, heatmaps, confidence interval bar charts, 
distribution histograms with manual Gaussian KDE, Q-Q normality plots, and per-mode OLS regression — all implemented from scratch using NumPy without statistical 
libraries. Built Linear Regression and XGBoost models to forecast daily passenger load, translating predictions into an operational bus deployment plan.
